Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Распологается слева
  • Блок навигации наполите контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседе равным .2em, при помощи margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседе равным .2em, при помощи margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для этой картинки

Нижни блок, подвал. Занимает всю доступную ширину. Высота опредедяется помещенным внутрь контентом.
